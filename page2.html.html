Мова HTML

Історія мови

Тім Бернерс-Лі — учений, який у 1991 році запропонував мову HTML для створення гіпертекстових документів у мережі Інтернет.

У 1994 році Тім Бернерс-Лі заснував консорціум W3C (World Wide Web Consortium), який займається стандартизацією технологій Інтернету.

Мова HTML постійно оновлюється, з'являються нові версії. Сучасною версією є HTML5.

HTML — це мова розмітки гіпертексту, що використовується для створення веб-сторінок.
