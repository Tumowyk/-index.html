Мова HTML

Історія мови

Один із творців HTML — Тім Бернерс-Лі. У 1991 році він опублікував опис мови HTML, яка призначалася для обміну науковою інформацією. З часом HTML став основою для створення вебсторінок.

Основні поняття

Тег — це команда браузеру, яка вказує, яким чином форматувати текст.

Контент — це текст чи зображення, що розміщується між тегами.
